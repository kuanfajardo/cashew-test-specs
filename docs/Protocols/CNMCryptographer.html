CNMCryptographer Protocol Reference

Conforms to NSObject
Declared in CNMCryptoServices.h

Overview

Protocol used for encrypting/decrypting string messages. Internally, the SDK has an interface very similar to this protocol.

If you wish to override provided crypto functionality create you would create an object that implements all these methods and provide SDK access to said object
