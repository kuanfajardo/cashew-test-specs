CNMMessageDefaultsServices Class Reference

Inherits from NSObject
Declared in CNMMessageDefaultsServices.h

Overview

This class provides the ability to set / get defaults for sending a message. If you call send message without the explicit attributes parameters, what is stored in defaults here is what will be used.

Defaults are: Priority : Low Ack Required : NO Forward Restricted : NO Expiration Duration : -1 (Meaning no expiration)

WARNING: Calling any of these methods before calling startServiceForCashewAccount:user:withCompletion:serviceInteruption: on CNMMessagingServices will cause your application to crash. This class needs to ensure you are successfully logged in and running messaging services
