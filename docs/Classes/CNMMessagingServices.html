Inherits from NSObject
Declared in CNMMessagingServices.h

Overview

Public interface for SDK user, allowing them to send and receive messages, create groups, create contacts, and all other things messaginging related.

This API is only useful after you successfully authenticate with Loment server in the UserServices class and the user has selected a CNMCashewAccount they wish to start messaging with.

After that has been achieved call startServiceForCashewAccount:user:withCompletion:serviceInteruption to begin all messaging services. When you are done, be sure to clean up by calling stop.

See sample project for example

Note: All other messaging API calls will fail to work until you successfully start services.
